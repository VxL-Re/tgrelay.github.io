How to get your Telegram session

  1. On a device where Telegram works, install Telethon: 
    pip install telethon
  2. Get API credentials from my.telegram.org
  3. Run this Python script: 
    from telethon import TelegramClient
from telethon.sessions import StringSession

API_ID = 12345  # your api_id
API_HASH = "abc123"  # your api_hash

async def main():
    client = TelegramClient(StringSession(), API_ID, API_HASH)
    await client.start()
    print("Session string:")
    print(client.session.save())

import asyncio
asyncio.run(main())
  4. Copy the printed session string
  5. Paste it on the setup screen

The session is like your login — keep it secret. It never leaves your browser except when you choose to save it as a GitHub Secret.
